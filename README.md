# CSL7110 – Machine Learning with Big Data

This repository contains the complete solution for the **CSL7110: Machine Learning with Big Data** assignment.  
The assignment demonstrates practical implementation of **Hadoop MapReduce** and **Apache Spark** for large-scale text processing and analysis.

---

## Repository Overview

This repository includes:

- **Answer_Images**  
  Contains screenshots of Hadoop and Spark outputs used as evidence in the assignment.

- **WordCount.java**  
  Hadoop MapReduce WordCount implementation for the Hadoop section of the assignment.

- **spark_assignment.py**  
  Spark program for metadata extraction from Project Gutenberg books (Question 10).

- **spark_q11.py**  
  Spark program implementing TF-IDF vectorization and cosine similarity to identify similar books (Question 11).

- **spark_q12.py**  
  Spark program for constructing and analyzing an author influence network (Question 12).

- **M25CSA033_ML_Big_DATA_Final.pdf**  
  Final assignment report containing written answers, explanations, and output screenshots.

- **README.md**  
  Description of the repository and assignment contents.

---

## Assignment Coverage

### Hadoop (Q1–Q9)
- Execution of the WordCount example using Hadoop MapReduce  
- Understanding mapper and reducer behavior  
- Use of HDFS commands  
- Explanation of replication factor  
- Performance-related configuration analysis  

### Apache Spark (Q10–Q12)
- **Q10:** Metadata extraction (title, release year, language, encoding) from Project Gutenberg books  
- **Q11:** TF-IDF vectorization and cosine similarity for identifying similar books  
- **Q12:** Author influence network analysis using in-degree and out-degree
